    🧪 Laboratorio Local — Prueba sin conexión a AWS
    
      LocalStack + Docker Compose
    
    
    
    
      
    
        Levanta un entorno local que emula los servicios AWS (S3, Secrets Manager, SQS, SSM)
        usando LocalStack. La app corre contra estos servicios locales sin necesitar
        credenciales de AWS. Ideal para que el equipo de desarrollo pruebe la migración en su PC.
      
    
    
    

    
    
      Analiza un artefacto Java primero para generar el entorno LocalStack
    
    

    
    
      
      
    
        
    
          
    docker-compose.localstack.yml
    
          
    
            
            
          
    
        
    
        
    
      
    

      
      
    
        
    ▶ Cómo levantar el laboratorio
    
        
    
          
    
            
    PASO 1 — Instalar LocalStack CLI
    
            pip install localstack && localstack start
          
    
          
    
            
    PASO 2 — Levantar el stack completo
    
            docker-compose -f docker-compose.localstack.yml up -d
          
    
          
    
            
    PASO 3 — Verificar que la app responde
    
            curl http://localhost:8080/actuator/health
          
    
          
    
            
    PASO 4 — Verificar servicios AWS emulados
    
            aws --endpoint-url=http://localhost:4566 secretsmanager list-secrets
